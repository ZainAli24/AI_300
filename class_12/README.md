## Python Two Dev Dependenies:
Bilkul simple alfaaz mein samjho 👍

## 🧹 Ruff

👉 **Ruff = code cleaner + mistake finder**

* Tumhara Python code check karta hai
* Batata hai:

  * kaha error hai
  * kaha code ghalat style mein hai
* Bohot fast hai ⚡

### Easy line:

👉 “Ruff tumhare code ko saaf aur sahi banata hai”

---

## 🧠 Pyright

👉 **Pyright = type checker (pehle hi error pakad leta hai)**

* Check karta hai ke:

  * variable ka type sahi use ho raha hai ya nahi
* Run karne se pehle hi error bata deta hai

### Example:

```python
name: str = 123  # ❌ wrong
```

👉 Pyright bolega: “ye string nahi hai”

---

## 🔥 Simple difference:

* **Ruff** → code style + common errors
* **Pyright** → type errors (data ka type sahi hai ya nahi)

---

## 🧩 Ek line mein dono:

👉 Ruff = “code theek likho”
👉 Pyright = “data theek use karo”
