# Clarify:
**Clarify Phase – Bilkul Simple, Beginner Level Samjh (English + Urdu Mix)**  

---

## **Clarify Ka Matlab Sirf Yeh Hai:**

> **"Spec mein jo cheez clear nahi hai — AI uske bare mein sawal poochhega, tum jawab do — spec aur clear ho jayega."**

---

### **Command:**  
```bash
/sp.clarify
```

---

### **Kab Use Karna Hai?**  
Jab tumhara **Specify** ho gaya ho, lekin lagta ho kuch **missing** ya **confusing** hai.  
Yeh **optional** hai — agar spec perfect lage to skip kar sakte ho.

---

### **AI Kya Karega? (Agent Ka Kaam)**

| AI Poochhega | Example (Calculator ke liye) |
|--------------|-------------------------------|
| **"Recent" ka matlab kya?** | → "Last 10 calculations ya last 30 days?" |
| **"Error message" kaise dikhega?** | → "Red color mein? Ya popup mein?" |
| **"Dark mode" kaise on hoga?** | → "Button se? Ya auto detect?" |
| **"Invalid input" kya hai?** | → "5++3 ya 5/0?" |

---

### **Tumhara Kaam Sirf Jawab Dena Hai**

| AI Ka Sawal | Tumhara Jawab (Simple) |
|-------------|------------------------|
| "History mein kitne calculations save karne hain?" | → **"Last 10."** |
| "10 ÷ 0 pe kya message dikhega?" | → **"Can't divide by 0" red mein.** |
| "Keyboard se C button kaam karega?" | → **"Haan, C ya Backspace."** |

---

### **Phir Kya Hoga?**
1. AI tumhare jawab se **spec.md** file ko **update** karega  
2. Sab cheez **100% clear** ho jayegi  
3. Ab **planning** (`/sp.plan`) bilkul perfect banega

---

### **Real Example (Tumhare Calculator ke liye)**

```bash
/sp.clarify
```

**AI Poochhega (Possible Questions):**

1. "History panel mein last 10 calculations — scrollable hona chahiye ya fixed?"
2. "Dark mode toggle — moon icon ya switch?"
3. "Error message kitne seconds tak dikhe?"
4. "Keyboard se 'Enter' = ke liye ya '=' dono?"
5. "Mobile pe buttons kitne bade hone chahiye?"

**Tum Jawab Do (Example):**
```
1. Scrollable
2. Moon/sun icon
3. 2 seconds
4. Dono
5. Minimum 50px height
```

**Done!** → Spec ab **crystal clear** hai.

---

### **Clarify Kyun Zaroori Hai?**

| Bina Clarify | Saath Clarify |
|--------------|---------------|
| Plan mein confusion | Plan 100% accurate |
| Bugs baad mein | Bugs pehle hi pakde |
| AI galat samjhega | AI bilkul sahi banayega |

---

### **Ab Tum Kya Karo?**

1. **Chat mein yeh paste karo:**
   ```bash
   /sp.clarify
   ```
2. **AI sawal poochhega → tum simple jawab do**
3. **Spec update ho jayega**
4. **Phir chala do:** `/sp.plan`

---

**Ek Line Mein Samjho:**

> **Clarify = AI bolta hai: "Yeh clear nahi, batao?" → Tum bolte ho: "Aise!" → Spec perfect!**
