# **AI in Development - 3 Levels:**

> ### **Level 1: AI Assistant Development**
> Is level mein AI ek helper ki tarah kaam karta hai jo development mein assist(help) karta hai. Jaise:
> - **Autocompletion**: AI next line ka code suggest karta hai
> - **Auto Bug Fixes**: Agar koi error aye toh AI usko automatically fix kar deta hai
> 
> Yani AI sirf hamare code likhne mein help kar raha hai, actual development hum khud kar rahe hain.
>

> ### **Level 2: AI (spec)Driven Development**
> Is level mein AI hamare partner ki tarah kaam karta hai. Hum sirf apni requirements aur specifications define karte hain ke hamein kya banana hai, aur AI pura application develop kar deta hai.
> 
> **Examples**: Claude Code, Gemini CLI, Codex, Qwen CLI
> 
> Ye coding agents hamare liye complete code likhte hain based on requirements.
>

> ### **Level 3: AI Native Software Development**
> Is level mein AI sirf development tool nahi balke product ka core part hai. Matlab:
> - Start se hi AI ko product mein integrate kar ke chalna hai
> - AI khud product ka feature hai, sirf development tool nahi
> - Example: Ek AI agent jo doosre AI agents banaye (AI building AI)
> 
> Yani humara product hi AI-powered hai aur development bhi AI se ho rahi hai.
>
